# Fitness Tracker Web App

A simple, browser‑based fitness tracker built with **HTML, CSS, and JavaScript**.  
All data is stored locally in your browser using `localStorage`, so no external database or account setup is required.

## Features
- 📅 **Calendar view**: Select dates and see which days have logged data.
- ⚖️ **Weight & Steps tracking**: Save daily entries and view history in a table.
- 💧 **Water tracker**: Set a custom goal and log water intake with quick +250ml / -250ml buttons.
- 🍽️ **Meals & macros**: Track calories, protein, carbs, and fats.
- 🔄 **Deficit propagation**: Apply calorie deficit ranges from a chosen date.
- 📊 **Export/Import JSON**: Backup or restore your data easily.
- 🧹 **Clear All**: Reset everything when needed.
- ✅ **Habit upkeep**: Mark streaks and reset progress.
